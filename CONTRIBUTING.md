# Contributing to fastapi-with-elk

Thanks for your interest in contributing! We welcome contributions of all kinds: bug reports, features, documentation improvements, and tests.

How to contribute

1. Search existing issues and pull requests for duplicates.
2. Open an issue using the appropriate template (bug or feature request) if needed.
3. Create a branch from `main` or `feature/*` named something like `feature/your-description`.
4. Follow the project's code style and add tests for new behavior.
5. Open a Pull Request using the PR template.
