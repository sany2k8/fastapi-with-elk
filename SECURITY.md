# Security Policy

If you believe you've found a security vulnerability, please report it privately to the maintainer at sany2k8@gmail.com with subject line "Security vulnerability report: fastapi-with-elk". Please include steps to reproduce, impact, and any relevant logs or exploit code.

We will respond as soon as possible and coordinate disclosure and remediation. If you prefer encrypted communication, please use a GPG-encrypted email (public key not provided here).

Public reporting

If you are unable to report privately, you may open an issue but please do not include exploit code or sensitive data in a public issue.
