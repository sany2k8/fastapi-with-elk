# Code of Conduct

We expect everyone who contributes to the project to abide by this Code of Conduct. We aim to provide a welcoming and inclusive environment for all contributors.

Reporting

If you experience or witness unacceptable behavior, please report it to the project maintainers at sany2k8@gmail.com.

Be respectful, courteous, and considerate in interactions. Any harassment, hateful language, or other behavior that makes others feel unwelcome will not be tolerated.

This project adopts the Contributor Covenant v2.1 as its Code of Conduct. For the full text, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
